# Namn på projecten: Fast Check
# Namn på deltagare: Viktoriia G, Emil J, Bilguun O, Promise OJ, Adham A, Maximilian M
### tävlar i kategori: Bästa helhetslösning
## Project och teknisk-beskrivning: Vi har skappat en webbsida som har en kalender som ger en möjlighet att skriva in dinna upkommande prov eller upgifter som man kan se när man hover över den. Vi har ochså skapad en merit räknare som låter dig att räkna ut din merit bäråande på dina kusrser och betig inom det ämnet. Våra lösning är för när man har inte nåfonstans att spara vitiga dattup för prov, inlämingar och för att se hur du ligget till i olika ämnä för att nå det merit du vill ha eller bähöver för det framtida utbildningen du vill söka.

## Extern producerad: vi har använd HTML, CSS, JavaSript och vi har laddat ner en foto för våran velkomnings sida
